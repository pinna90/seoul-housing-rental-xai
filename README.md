# Determinants of Housing Rental Prices in Seoul: Applying Explainable AI

This repository contains the official code accompanying the published paper:

> **Kim, T.-Y., Park, E., & Ryu, D. (2025).**
> *Determinants of Housing Rental Prices in Seoul: Applying Explainable AI.*
> **Spatial Economic Analysis, 20(2), 312–332.**
> https://doi.org/10.1080/17421772.2024.2418906

## Authors

* Tae-Young Kim
* Eunil Park
* Doojin Ryu

## Overview

This repository provides the implementation used in the published study on the determinants of housing rental prices in Seoul. The study applies multiple machine learning models and Explainable Artificial Intelligence (XAI) techniques to identify and interpret the factors influencing housing rental prices.

## Repository Structure

```text
.
├── code_for_SpatialEconomicAnalysis.ipynb
├── README.md
└── ...
```

The primary implementation is provided in:

* **`code_for_SpatialEconomicAnalysis.ipynb`**

Please refer to this notebook for the complete workflow, including:

* Data preprocessing
* Machine learning model development
* Model evaluation
* 10-fold cross-validation
* TreeSHAP-based Explainable AI (XAI) analysis
* Generation of the figures presented in the paper
